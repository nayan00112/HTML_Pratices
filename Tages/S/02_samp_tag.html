HTML <samp> tag

HTML <samp> tag is used tag which is used to represent the sample output from a computer programm or a script, in the browser.

The contant written between the <samp> and </samp>, tags render in browser's default monospace font.

Example:

code:

#include<iostream>
using namespace std;
int main()
{
    cout << "Hello World!!!" << endl;
    return 0;
}

output:

Hello World!!!
