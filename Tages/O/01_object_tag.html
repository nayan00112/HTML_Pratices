HTML object tag

HTML <object> tag is used to embeded muiltimedia files on webpages. The <object> tag can be include muiltimedia files such as video, audio, image, PDF, Java Applets, or another page on your page.

HTML <paramt> tag also used with <object> tag to pass parameters to plugin which has been included with <object> tag

If you insert text between the <object> and </object> tags, then it will only be displayed if the browser does not support the <object> tag
