HTML Canvas Tag

The HTML canvas element proides HTML a bitmapped surface to work with. It is used to draw graphics on the webpage.

The html 5 canvas tag is used to draw grafics using scripting language like javascript.

The canvas element is only a container for grafics, you must need a scripting language to draw the grafics. The camvas element allows for dynamic and scriptable rendering of 2D Shapes and bitmap images.

It is a low level, procidual model that update a bitmap and dose not have a built-in scene. There are several methods in canvas to draw path, boxes circle, text and add images.
