HTML NoScript tag

HTML <noscript> tag is used to definr an alternative content fot the user who have disabled scripting from the browser and want to access to the web page.

The <noscript> element can be used within <head> and <body> tags.

While using noscript tag inside <head> elememt, <noscript> mnust contain <link>, <style> and <meta> tags.

The text inside the <noscript> element will be display if the user's browser is not script supporting.
