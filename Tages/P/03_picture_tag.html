HTML <picture> tag

HTML <picture> tag is used in responsive web designing where we need to load the different images based on their viewpoint, hight, width, orientation and pixel density.

The <picture> tag contains one or more <source> element and one <img> element.

According to the viewport, the matching image will be loaded from different <source> tag, and if no source contains the matching image, then the default image present in <img> tag will be displayed on the browser.

The tag is new tag in html5
