Meta Tag

HTML <meta> tag is used to represent the metadata about the html document. It specifies page description, keyword, copyright, language, author of the documents, etc.

The metadata does not display on the webpage, but it is used by search engines, browsers and other web services which scan the site or webpage to know about the webpage.

With the help of meta tag, you can experiment and priview that how your webpage will render on the browser.

The meta tag is placed within the head tag, and it can be used more then one times in a document.

Syntax:

<meta charset="utf-8">

Following are some specification about the HTML <meta> tag:

  • Display: None
  • Start tag/End tag: Empty Tag (Only Start Tag)
  • Usage: Document Structural

Following are some specific syntex of meta tag which shows the different uses of meta Tag.

  1. It defined the character encoding. The value of charset is "utf-8" which means it will support to display any language.
    2. <meta charset="utf-8">
  2. It is specifies the list of keyword which is used by search engines.
    3. <meta name="keywords" content="HTML, CSS, Javascript">
  3. It defined the website description which is useful to provide relevant search performed by search engines.
    4. <meta name="description" content="Free Online Documentations">
  4. It specifies the author of the page. It is useful to etract author information by contex management system automatically.
    5. <meta name="author" content="thisAuthor">
  5. It specifies to provide instruction to the browser to automatically refresh the content after every 50 sec. (or any given time).
    6. <meta name="refresh" content="50">
  6. In this below Example we have set a URL with content so it will automatically redirect to the given page after the provided time.
    7. <meta http-equiv="refresh" content="5;url=https://www.youtub.com/">
  7. It specifies the view to control the page dimension and scalling so that our website looks good on all devices. If this tag is present, it include that this page is mobile devices supported.
    8. <meta name="viewport" content="width=device-width, initial-scale=0.1">
