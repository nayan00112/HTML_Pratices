HTML dir tage (Not supported in HTML5)

HTML dir tag was as a container for the list of files/folders or content of the directory.
The dir element is used with the li tags, and list of directories renders in bullets by default.

Note: Do not use HTML dir tag as it is deprecated now and not supported in HTML5.
you can use ul tag and CSS property insted.
